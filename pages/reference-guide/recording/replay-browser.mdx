# Replay Browser

The Replay Browser is software you [download and install](/getting-started/bug-reports/installing-the-replay-browser) on your computer to record replays. To learn more about recording a replay with the browser, [check out our guide here](/getting-started/recording-your-first-replay).

**The Replay Browser is not required to view existing replays.**

</Callout>

## Browser Support

- Firefox on Mac, Linux, and Windows
- Chromium for Linux is in Beta

# Replay Launcher Chrome Extension

The [Replay Launcher Chrome Extension](https://chrome.google.com/webstore/detail/replay-launcher/djjkbhpjjnhbebpcdkjiggjloabpkkeo?hl=en) lets you quickly open the Replay Browser from Chrome. Click the extension to open your current URL in the Replay Browser and immediately start recording.

## Getting Started

1. Install the extension from the [Chrome Web Store](https://chrome.google.com/webstore/detail/replay-launcher/djjkbhpjjnhbebpcdkjiggjloabpkkeo/related?hl=en)
2. Click the puzzle icon in your browser toolbar and pin the extension so it‚Äôs always available when you need it.
    
![Screen Shot 2022-04-28 at 6.01.06 PM.png](/images/Screen_Shot_2022-04-28_at_6.01.06_PM_raaht.png)
    
3. Whenever you are ready to record, click the Replay extension icon in the browser toolbar, then select ‚ÄúRecord this website.‚Äù **Note: The Replay Browser must be installed before recording**. You can also install the browser from the extension.
4. The first time you record a new site, the browser will prompt you to allow opening the browser. Select the checkbox to prevent this step next time, then click ‚ÄúOpen Replay‚Äù.
    
![Screen Shot 2022-04-28 at 12.24.28 PM.png](/images/Screen_Shot_2022-04-28_at_12.24.28_PM_bprfw.png)
    
5. The Replay Browser will launch to the exact URL already open in Chrome and begin recording automatically. From here, [record](/getting-started/recording-your-first-replay) and [share](/getting-started/bug-reports/sharing-your-replay) as you normally would!
